## 🛠️ Technikai stósz (Tech Stack)

Az oldalt a modern alapokra húztam fel:
- **HTML5:** Szigorúan szemantikus felépítés (`<article>`, `<section>`, `<header>`, stb.). Semmi felesleges `div`!
- **CSS3:** Egyedi "Cyber-Gold" téma, lekerekített sarkok, és zöld `code` box.
- **Bootstrap 5.3:** Reszponzivitás és profi formok.
- **YouTube API:** Zökkenőmentes videó beágyazás a legjobb zenével a világon.
